## Open Source Releases
- [aig-guardian 0.8.0](https://pypi.org/project/aig-guardian/0.8.0/) — Zero‑dependency middleware that spots prompt injections, jailbreaks, and PII leaks while covering the OWASP LLM Top 10. Plug it into FastAPI, LangChain, or direct API calls for a quick security layer. 🛡️  
- [sam-gate 0.1.3](https://pypi.org/project/sam-gate/0.1.3/) — A semantic‑aware memory gate that compresses heterogeneous KV‑caches in transformers, aiming to cut inference memory without hurting accuracy. Drop‑in module for existing model serving pipelines. 📦  
- [crewai 1.14.0a3](https://pypi.org/project/crewai/1.14.0a3/) — Framework for orchestrating role‑playing, autonomous AI agents that collaborate on complex tasks. This alpha tightens agent communication and task handling for smoother multi‑agent workflows. 🤖  
- [toolregistry-hub 0.8.0](https://pypi.org/project/toolregistry-hub/0.8.0/) — Utility belt for LLMs offering math, date/time, file ops, web search, unit conversion, and more. Handy when you need ready‑made tools for function calling or general scripting. 🧰  
- [torch-mpo 0.2.3](https://pypi.org/project/torch-mpo/0.2.3/) — PyTorch implementation of Matrix Product Operators for compressing large weight matrices into low‑rank forms. Supports training and inference with lower memory and compute footprints. 📉  
- [nteract 2.1.1a202604060758](https://pypi.org/project/nteract/2.1.1a202604060758/) — Brings AI capabilities to Jupyter notebooks via an MCP server that talks to Claude, ChatGPT, Gemini, OpenCode, and any agent. Enables inline AI‑assisted coding and data exploration. 📓  

## AI Dev Tools
- [Gemma: Google DeepMind's Open-Weight LLM Library](https://github.com/google-deepmind/gemma) — Provides access to DeepMind’s open‑weight LLMs with inference and fine‑tuning code. Built for efficiency so you can experiment locally without massive hardware. 🧠  
- [RAG-Anything: All-in-One Retrieval-Augmented Generation Framework](https://github.com/HKUDS/RAG-Anything) — Modular RAG pipeline that lets you swap loaders, embedders, stores, and LLMs via config. Reduces boilerplate when building custom retrieval‑augmented apps. 🔍  
- [Pi-Mono: AI Agent Toolkit with Coding Agent CLI and LLM API](https://github.com/badlogic/pi-mono) — Full‑stack kit for AI agents: coding agent CLI, unified LLM API, TUI/web UIs, Slack bot, and vLLM pod support. Aims to cut the glue code needed for autonomous agents. 🛠️  
- [ByteRover CLI: Portable Memory Layer for Autonomous Coding Agents](https://github.com/campfirein/byterover-cli) — Persistent storage for agent experiences and knowledge, usable across sessions and environments. Helps agents retain long‑term context without rebuilding from scratch. 💾  
- [QMD: Mini CLI Search Engine for Local Docs and Knowledge Bases](https://github.com/tobi/qmd) — Offline, embedding‑based search tool for your personal docs, notes, and wikis. Fast, relevant results with zero external dependencies. 🔎  

## MCP Servers & Integrations
- [Google Calendar MCP Server](https://smithery.ai/servers/googlecalendar) — Lets AI agents create events, check free/busy, set reminders, and manage calendars across time zones via the Model Context Protocol. Simple way to give agents scheduling superpowers. 📅
