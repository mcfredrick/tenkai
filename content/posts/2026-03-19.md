## Open Source Releases
**[pyodide](https://github.com/pyodide/pyodide)** — Python distribution for browsers/Node.js via WebAssembly, enabling NumPy/pandas/scikit-learn client-side execution for notebooks and serverless ML.

**[nvidia/NVIDIA-Nemotron-3-Super-120B-A12B-BF16](https://huggingface.co/nvidia/NVIDIA-Nemotron-3-Super-120B-A12B-BF16)** — 120B parameter MoE language model in BF16, supporting European/Asian languages with Azure deployment.

**[unsloth](https://github.com/unslothai/unsloth)** — Unified UI for training/fine-tuning open-weight LLMs (Qwen/DeepSeek) with LoRA quantization, targeting developers avoiding vendor lock-in.

**[newton](https://github.com/newton-physics/newton)** — GPU-accelerated physics simulation engine for robotics, offering rigid/soft-body dynamics and integration with control pipelines.

**[open-swe](https://github.com/langchain-ai/open-swe)** — Asynchronous coding agent framework automating bug fixes, feature impl, and code reviews via LLM orchestration.

**[chatterbox](https://github.com/resemble-ai/chatterbox)** — High-fidelity open-source TTS system supporting multilingual real-time speech synthesis.

**[MaxKB](https://github.com/1Panel-dev/MaxKB)** — Enterprise agent platform for knowledge retrieval, tool use, and multi-agent orchestration with easy deployment.

**[honcho](https://github.com/plastic-labs/honcho)** — Memory library for stateful AI agents, enabling persistent storage/retrieval with vector search.

**[letta-code](https://github.com/letta-ai/letta-code)** — Memory-first coding agent retaining context across sessions for code generation/debugging.

**[GitNexus](https://github.com/abhigyanpatwari/GitNexus)** — Client-side code intelligence engine generating interactive knowledge graphs from GitHub repos.

**[Jackrong/Qwen3.5-27B-Claude-4.6-Opus-Reasoning-Distilled](https://huggingface.co/Jackrong/Qwen3.5-27B-Claude-4.6-Opus-Reasoning-Distilled)** — Distilled reasoning model emulating Claude 4.6 Opus capabilities for logical tasks.

**[Tesslate/OmniCoder-9B](https://huggingface.co/Tesslate/OmniCoder-9B)** — Image-to-text-to-text coding agent for conversational code assistance.

**[Lightricks/LTX-2.3](https://huggingface.co/Lightricks/LTX-2.3)** — Video generation suite supporting text/image/audio-to-video transformations.

**[nvidia/NVIDIA-Nemotron-3-Super-120B-A12B-NVFP4](https://huggingface.co/nvidia/NVIDIA-Nemotron-3-Super-120B-A12B-NVFP4)** — 8-bit NVFP4 quantized variant of Nemotron 3 Super for efficient inference.

**[nvidia/NVIDIA-Nemotron-3-Super-120B-A12B-FP8](https://huggingface.co/nvidia/NVIDIA-Nemotron-3-Super-120B-A12B-FP8)** — FP8 precision version optimized for high-throughput inference with Azure deployment.

**[Draft-and-Prune: Improving the Reliability of Auto-formalization for Logical Reasoning](https://arxiv.org/abs/2603.17233)** — Two-stage auto-formalization pipeline drafting and pruning incorrect logical statements.

**[Contrastive Reasoning Alignment: Reinforcement Learning from Hidden Representations](https://arxiv.org/abs/2603.17305)** — CRAFT method using contrastive learning to steer LLMs toward safe reasoning traces.
